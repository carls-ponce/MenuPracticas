Web 1.0
  • Se publica la primera página web creada por Tim Berners-Lee usando un computador NeXT
  • Nacen los primeros navegadores web, liderandolo Mosaic
  • Una de las primeras empresas en proveer este servicio de manera masiva fue Hotmail.
  • Las marcas comenzaron a crear sus páginas web, enfocadas netamente en comunicar información básica sobre productos y servicios.
  • Etiquetas HTML patentadas por los navegadores, esto ocasionó problemas de compatibilidad
  • Las páginas eran puramente estáticas
  • El contenido era unidireccional, la información se generaba para un grupo de consumidores que simplemente la consumían.
  • uso de framesets o marcos.
  • Extensiones propias del HTML como blink y marquee, etiquetas introducidas durante la guerra de navegadores web.
  • Libros de visitas en línea o guestbooks.

Web 2.0
  1. El término fue utilizado por primera vez por Darcy DiNucci en 1999, en su artículo "Fragmented future",13​ aunque no fue hasta 2004 cuando Tim O'Reilly lo hizo popular.
  2. La Web 2.0 es una segunda generación de servicios basados en la Web, que enfatiza en la colaboración online, la conectividad y la posibilidad de compartir contenidos entre los usuarios.
  3. Denominada también "web social", brinda diversas tecnologías de participación.
  4. Los usuarios dejan de ser usuarios pasivos para convertirse en usuarios activos que participan y contribuyen en el contenido.
  • La información es lo que mueve el internet
  • Nacen los blogs, redes sociales, wikis; aunque no como las conocemos hoy en día.
  • Funcionalidad que permite añadir metadatos
  • La web 2.0 ofrece a casi todos los usuarios la misma libertad para colaborar.
  • El botón «me gusta» y la calificación de contenidos (que permite al usuario expresar su opinión sobre el contenido) y los marcadores sociales forman parte de las características de la web 2.0.

Web 3.0
  • Esta nueva versión de Internet se encuentra estrechamente ligada con el concepto de 'Web Semántica'.
  • El primer paso hacia la "Web 3.0" es el nacimiento de la "Data Web.
  • Internet alcanzo los mil cien millones de usuarios.
  • nternet tiene 634M de sitios web de comercio y ventas que superan los 100 mil millones de dólares a nivel global.
  • Las aplicaciones móviles representaron mas de la mitad del tiempo de uso y visualización de contenidos digitales.
Búsquedas Inteligentes:
La web 3.0 busca crear un nuevo sistema de clasificación de páginas web estrechamente ligado a las necesidades y características de los usuarios.
Más rapidez:
Las principales operadoras de telecomunicaciones han implementado conexiones de banda ancha para garantizar una experiencia de uso más rápida.
Espacios tridimensionales:
Los usuarios pueden acceder a nuevas formas de visualizar la web, con espacios tridimensionales. Un claro ejemplo de esto es Google Earth.
Computación en la nube
Con la creación de nuevos espacios de almacenamiento, no sólo de datos sino de programas, la web se convierte en un espacio ejecutable a modo de computador universal.

Web 4.0
  1. Se potencia la web semántica con el aporte de la inteligencia artificial para así proveerle a los consumidores una mejor experiencia.
  2. La comprensión del lenguaje y la capacidad de procesar una orden o recomendación serán unos de los elementos mas importantes.
  3. La web 4.0 será mucho más activa y funcionará como un asistente virtual con inteligencia artificial.
  4. La web tendrá un comportamiento más inteligente y predictivo, un conjunto de acciones que analizan y razonan cada mota de información, comunicación máquina a máquina (M2M) y aprendizaje profundo.
  5. Las criptomonedas y bancos digitales se hacen más presentes. En la Web se imprime, se acuña y se gestiona, gracias a las criptomonedas. Cada transacción conlleva una ID única y cada moneda (virtual) tiene nombre y apellido: Internet es ahora un gran libro contable.
  • Se centra en ofrecer un comportamiento más inteligente y más predictivo, de modo que podamos, con sólo realizar una afirmación o una llamada, poner en marcha un conjunto de acciones que tendrán como resultando aquello que pedimos, deseamos o decimos.
  • Nuevos modelos de comunicación máquina-máquina (M2M).
  • Uso de información de contexto del usuario. Sentiment analysis, geolocalización, sensores…
  • Nuevo modelo de interacción con el usuario.
